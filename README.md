# 제목: 블록체인의 신뢰 형성 구조와 기존 금융 시스템의 신뢰 구조 비교

## 부서명: 디지털서비스개발부
## 이름: 신승선 프로

-------------------

## 레포트 내용
기존 금융 시스템은 우리회사와 같은 시중은행, 한국은행과 같은 중앙은행, 금융감독원/금융위원회와 같은 정부기관 등 “중앙기관”의 권위와 보증에 전적으로 의존한다. 그렇기에 거래가 정상적으로 이루어질 것이라는 신뢰는 “은행이 거짓말을 하거나 실수하지 않을 것”이라는 제3자인 중앙기관의 신용으로부터 나온다.
반면에 블록체인에서는 “암호학”과 “네트워크 참여자들의 합의”를 통해 신뢰를 만들어낸다. 이를 Trustless(신뢰가 필요 없는) 시스템이라고 부르는데, 사람이나 특정 기관을 믿을 필요 없이 시스템 코드와 수학에 의해 규칙적으로 작동하기 때문이다.
